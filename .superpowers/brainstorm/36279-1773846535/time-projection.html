Option C — Both: Chart + Table + Narrative

Full analysis section combining the chart for visual impact, the table for detail, and the narrative for copy-paste into resource requests. Most comprehensive but takes more vertical space.

Projection model

  • Baseline: Weighted average of last 3-6 months (recent months weighted more heavily)
  • Renewal spike: 1.5-2x multiplier for months with renewals, based on historical pattern during prior renewal months
  • Cost-to-serve ratio: (annual revenue) / (annual effort cost at configurable hourly rate)
  • Narrative: Auto-generated text summarizing the projection, suitable for copy-paste into internal resource requests
